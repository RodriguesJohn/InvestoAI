# Investo AI Dashboard

A React implementation of the Investo AI dashboard design from Figma.

## Design Source

- **Figma**: [Investo AI](https://www.figma.com/design/IhcEc1v30hNBE13wltcsbJ/Investo-AI?node-id=119-1626)
- **Node**: Dashboard (119-1626)

## Features

- **Sidebar**: Navigation with Dashboard, Investments, Goals, Learn, Settings
- **KPI Cards**: Total Value, Today's Change, Monthly Return
- **AI Allocation**: Pie chart showing portfolio allocation (Stocks, Bonds, Cash)
- **Top Holdings**: Apple Inc., Tesla Inc., S&P 500 ETF with values
- **Goals Progress**: Grow Wealth, Emergency Fund, Retirement with progress bars
- **Checklist**: "How we'll manage your money" with check items
- **Tip of the Day**: AI-powered recommendations carousel
- **Add Funds Button**: Primary CTA in header

## Tech Stack

- React 19 + TypeScript
- Vite 7
- Tailwind CSS 4
- Lucide React (icons)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the dashboard.

## Build

```bash
npm run build
```
